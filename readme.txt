== Description ==

Agrega Yape, Plin, Tunki y Lukita como formas de pago en tu tienda WooCommerce. Recibe pagos sin comisiones desde Yape, Plin, Tunki y Lukita.

== Installation ==

1. Carga el plugin dentro de la carpeta `/wp-content/plugins/` o instala desde la lista de plugins
2. Activa el plugin desde la opcion 'Plugins' en el menu de WordPress
3. En los ajustes del WooCommerce encontrarás todos los nuevos métodos de pago
4. En cada opción de Pago podrás agregar un QR de tu comercio.
